***** Kool *****
Theme Name: Kool
Description:  Kool is a simple, clean, fully responsive, magazine/blog style wordpress theme. It easily adapts and looks great on mobile, tab, desktop or any other devices. It is easily customizable, you can customize it following the guideline stated on http://hit-hut.com/kool/blog/ It has a front page template with its own widgets and an optional full-width, no-sidebar page template, a custom post widget to display featured image on sidebar, styling for post formats on both index and single views, custom menu, header image, background and much more.

================================= Installation ================================

1. Automatic Installation:
      Go to WP-Admin > Appearance > Themes > Install Themes > Upload and choose the theme zip folder.

Manual installation:

1. Unzip kool.Zip and Upload the `kool` folder to the `/wp-content/themes/` directory

Activiation and Use
.....................

1. Activate the Theme through the 'Themes' menu in WordPress 
2. Click Appearance to get different options.  


=============================== Supported Browser ================================

Mozilla, Chrome, Safari, IE 9,8,7
